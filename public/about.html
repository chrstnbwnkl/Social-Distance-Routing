• About the Project

What is this?

The COVID-19 pandemic has had an impact on human mobility, especially in dense urban areas. In order to minimize the risk of contagion, people are advised to maintain a distance of 1.5 meters to each other. This is not always an easy task, especially as a pedestrian in a crowded public space. The goal of this project is to find out how geo-information can contribute to minimizing this risk for pedestrians. The City of Melbourne serves as a great playground for this investigation, as there is an existing pedestrian count system that delivers near real time information on the number of pedestrians walking at different locations throughout the city's downtown area. The result of this project is a navigation application that uses this count data as an input for a routing network with a custom cost that is updated four times an hour.

Who are we?

We're a group of MSc Geo-information students at the University of Utrecht, the Netherlands. We have been building this application as part of an ongoing student project.
